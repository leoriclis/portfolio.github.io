Maquette Web

Au cours de ma 1ère année de BUT j'ai pu avoir comme matière du développement web initiant à la fabrication d'un site type maquette en HTML CSS


C'était pour moi une première approche de ces langages de programmation et m'a permis d'apprendre et d'aimer la première phase de création d'un site internet

Les règles :

Les règles étaient assez simples. Il fallait recréé un site web en maquette pour qu'elle soit la plus proche de ce qui était présent

Quelles-ont été les compétences travaillés ?

Comme j'ai pu le dire juste avant j'ai pu travaillé avec les langages HTML CSS et aussi Affinity Designer


L'utilisation d'affinity designer m'a permis d'avoir un premier visuel de ce qu'il fallait faire avant de coder pour éviter de me perdre dans un projet assez long


Ce site m'a permis d'apprendre beaucoup de compétences indispensable lors d'utilisation d'HTML ou CSS comme les classes ou bien les flexbox

Rendu

Pour un premier projet je suis assez satisfait du résultat sachant que c'était la première fois que j'utilisais ces langages de programmation ainsi qu'affinity designer Le rendu est très proche du site initial.
